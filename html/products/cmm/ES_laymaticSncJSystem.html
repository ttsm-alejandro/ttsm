Sistema Laymatic SNC J

Layout Machine Modelo J mejorado a sistema CNC


  • Fácil de utilizar
  • El Modelo J, con una especificación estándar muy simple, permite seleccionar entre una perilla adicional y un ajuste fino de acuerdo con la aplicación; En el Modelo J los elementos estan diseñados de una manera amigable para el usuario.
  • El grado de precisión esta establecido de acuerdo con el estándar industrial japonés JIS B7440 (Sin embargo, la precisión mostrada aplica sólo cuando se instala en una mesa de mediciones TTS).
  • Al usar perfiles cuadrados en los componentes de la máquina, se previenen cabeceo y balanceo
  • The arm head and column head are of type divided, making the adjustment easier; the true rectangularity is improved and the backlash hardly produced.
  • Can be used for multi purposes.
  • Able to use the conventional driving system and attachments.
  • Up-grading of the system is possible in order to use a manual mode drive /CNC.
  • The CNC system is improved in rigidity and precision by using carbon fiber-reinforced plastics (CFRP) arm
