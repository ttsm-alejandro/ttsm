ATS Bus

ATS Bus es una solución que puede crear vínculos entre el equipo y el sistema a corto plazo a un costo menor sin necesidad de programación. Es una herramienta inteligente para promover IoT en las fábricas que no requiere una habilidad de programación manual.

Qué puede hacer ATS Bus

■ Recopilación y visualización de datos de equipos
■ Seguimiento de productos en proceso
■ Conexión eficiente del sistema
■ Vinculación del sistema de construcción a corto plazo sin programación

Características del producto

Recopilación y visualización de datos de equipos

【Estructura general】

0. Preparación: creación de un mecanismo de recopilación y divulgación de datos

ATS Bus es una herramienta que crea un enlace de sistema entre nodos, como equipos, dispositivos de medición, sensores, etc., plataformas de IoT y bases de datos. Además del bus ATS, es necesario construir un mecanismo de ENTRADA / SALIDA para realizar IoT como se muestra en las figuras siguientes.

1. Funcionamiento del bus ATS: 1. Definición de datos

Defina las palabras clave de identificación y el contenido de los datos para los datos recopilados de cada equipo y nodo de medición, y los datos para enviar a una plataforma de IoT y una capa superior en la base de datos, y guárdelos con un mensaje de identificación.

2. Operación del bus ATS - 2. Definición de ruta e I / F

Defina la interfaz, el objetivo de monitoreo y el objetivo de transferencia para los datos recopilados de cada equipo y nodo de medición, y los datos para enviar a una plataforma de IoT y una capa superior en la base de datos, y guárdelos con el nombre del mensaje de identificación.

3. Operación del bus ATS - 3. Definición de eventos, acciones y vínculos

Vincular acciones como la transferencia de datos a las plataformas de IoT y la capa superior en la base de datos para eventos como la carga de datos de cada equipo y nodo de medición, y construir un control de datos entre los nodos.

4. Divulgación de datos con herramientas de BI

Los datos recopilados de cada nodo en los sitios de fabricación se recopilan en la base de datos o se envían al sistema de producción a través de ATS Bus. Cree una página de informe en el estilo que desee cada usuario con la información requerida y divulguela.
