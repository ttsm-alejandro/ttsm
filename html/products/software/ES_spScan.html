spScan

spScan es un software de ingeniería inversa para crear una superficie NURBS a partir de los datos de medición sin contacto.

Se puede crear una superficie con la calidad disponible en CAD / CAM / CAE, en un corto período de tiempo (incluido el procesamiento automático).

Más información: https://www.armonicos.co.jp/en/products/spscan/

Propósito

Medición de molde y marcha atrás

1. Cree datos maestros para la producción en masa.
Crear datos de molde CAD a partir de los datos de medición del molde (reverso parcial)

2. Cree una copia de datos de molde.

Medición e inversión del producto

1. Análisis y retroalimentación CAD del producto prototipo.
- Creación de una superficie simple para el diseño, como interferencia (ajuste), etc.
- Varias simulaciones como fuerza / vibración / fluido

2. Creación de datos de producto coincidente (producto pasado).

3. Análisis de productos de otras empresas (productos funcionales).

Característica

1. Definición de superficie de alta calidad para conectar con el mecanizado
2. Ajuste de la precisión a los datos de medición
3. Curvas de contorno de superficie que se pueden editar en CAD / CAM / CAE
4. Tamaño de datos ligero (orden y número de parches óptimos)
5. Superficie dividida libre (soporte para superficie recortada)
6. Procesamiento automático

Variación de la técnica inversa

Se proporcionarán cuatro métodos de creación de superficies.
Puede seleccionarse teniendo en cuenta el uso y las horas de trabajo.
