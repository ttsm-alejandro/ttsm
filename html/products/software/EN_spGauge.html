spGauge

Product Inspection Tool using Point Cloud Data measured by Non-contact digitizers and 3D-CAD Data

spGauge is to detect Errors of Mold shape, Product shape from Point Cloud Data and 3D-CAD Data. Furthermore spGauge can evaluate their Errors quantitatively.

Detection of Error Part quickly


spGauge compares Point Cloud Data and 3D CAD Data, and changes color of the error part in Point Cloud Data. So it is possible to detect them quickly.

Confirmation of Thickness & Spring-Back variation without cutting

It is possible to confirm Thickness by Error Map without cutting a Plastic Product, a Cast Product, and a Forged Product.
And On a Press Product, it is possible to confirm the variation of Spring-Back and Warp.
