Robot accuracy improvement solution / Robot calibration system


Mission

Shorten and simplify robot teaching tasks Increase robot accuracy and make it do more various types of work

Solution

Calibrating a robot and improving its accuracy is effective for simplifying teaching tasks, manufacturing, assembly and jig-free tasks, paint coating, transporting, and work involving measuring.

Proposal Contents

Leica Laser Tracker AT960(SR)/MR/LR/XR + T-Mac + SA Machine or RoboDyn

Using the Leica Laser Tracker system with dedicated software enables robot accuracy to be improved. The system can also support tasks such as alignment adjustments made by the user.
