Quality evaluation of 3D additive manufacturing products and 3D model manufacturing by reverse engineering


Mission

- Facing client’s requests to manufacture a product which has no CAD model
(Some of the requests require high accuracy since the product is structural parts or consists of many free-form surfaces.)

Solution

Use of reverse engineering
ZEISS COMET system is a camera-type non-contact scanner which scans 3D data by projecting stripe-pattern lights onto the object using a LED projector.

The machine starts measuring the object when it is put on a rotary table and easily scans the whole shape of the product with high accuracy. After this step, it can import the scanned data to the reverse engineering soft to create high-quality and high-accuracy CAD data and manufacture its copy with the 3D additive manufacturing device based on the imported CAD data.

It is also possible to compare the manufactured 3D copy with the original measurement data with the color map inspection function by measuring the copy with the COMET system.
