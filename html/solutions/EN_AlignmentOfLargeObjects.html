Alignment of large objects such as airplane doors.


Mission

Do 3D positioning of relatively large objects such as airplane doors. Usually large object accurate positioning is difficult and it also takes a great deal of time to perform this task. The position accuracy and work time also depends on the person doing the work.

Solution

MoveInspect system is a camera type CMM that can dynamically comprehend and analyze the behavior of a target object after calculating the spatial coordinate position of multiple targets affixed to the object being measured.
Establishing reference points (for reference) on the door frame enables the use of a dynamic reference function. Using this function enables the door to be aligned after measuring the horizontal and vertical position with very little influence from vibration, even in locations with vibration.
