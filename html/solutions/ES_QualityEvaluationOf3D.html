Evaluación de calidad de productos de fabricación aditiva 3D y fabricación de modelos 3D por ingeniería inversa


Mision

Hacer frente a las solicitudes del cliente para fabricar un producto que no tiene modelo CAD
(Algunas de las solicitudes requieren una alta precisión ya que el producto son piezas estructurales o consta de muchas superficies de forma libre).

Solucion

Uso de ingeniería inversa
El sistema ZEISS COMET es un escáner sin contacto tipo cámara que escanea datos en 3D proyectando luces con patrones de rayas sobre el objeto utilizando un proyector LED.

La máquina comienza a medir el objeto cuando se coloca sobre una mesa giratoria y escanea fácilmente toda la forma del producto con alta precisión. Después de este paso, puede importar los datos escaneados al software de ingeniería inversa para crear datos CAD de alta calidad y precisión y fabricar su copia con el dispositivo de fabricación aditiva 3D basado en los datos CAD importados.

También es posible comparar la copia 3D fabricada con los datos de medición originales con la función de inspección del mapa de color midiendo la copia con el sistema COMET.
